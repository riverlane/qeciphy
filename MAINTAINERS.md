## Maintainer Responsibilities

### Code Review
- Review and approve pull requests
- Ensure code quality and adherence to project standards
- Verify proper testing and documentation

### Issue Management
- Triage incoming issues and feature requests
- Provide technical guidance to contributors
- Maintain issue labels and milestones

### Release Management
- Coordinate releases and version tagging
- Maintain release notes and changelogs
- Ensure compatibility across supported platforms

### Documentation
- Review documentation updates
- Ensure technical accuracy
- Maintain integration guides and examples

## Governance

All decisions related to project direction, design, and releases are made by the maintainer(s) listed above. Future maintainers may be added by Riverlane Ltd.
